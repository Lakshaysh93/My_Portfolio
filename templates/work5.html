005 : Automation and Testing

A test case suite executed using specialised automated testing software tools as part of the software testing technique is known as automation testing. Additionally, the automation testing software may generate thorough test reports, compare expected and actual findings, and enter test data into the System Under Test.

"As ironic as it seems, the challenge of a tester is to test as little as possible. Test less, but test smarter."

Federico Toledo
